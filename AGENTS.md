# AGENTS.md

## プロジェクト概要

このプロジェクトは、Tanstack Start、Bun、Vite、TypeScript、Convexを活用した、型安全かつ高速なWebアプリケーションです。

## セットアップコマンド

- 依存インストール

  ```sh
  bun install
  ```

- 開発サーバー起動

  ```sh
  bun run dev
  ```

- ビルド

  ```sh
  bun run build
  ```

## コーディングスタイル

- TypeScriptは厳格な型付け（`any`禁止）
- コンポーネントは`function`宣言を使うこと
- その他の関数はアロー関数（`const`で宣言）を使うこと
- 関数型アプローチを推奨し、クラスは最小限にすること
- ディレクトリ名は小文字とハイフン(-)区切り(lowercase-with-dashes)にすること
- マークダウンは以下のスタイルにすること
  - 英語と日本語の間にスペースを入れないこと
  - 全角コロンは使わず、必ず半角コロン(:)を使うこと
  - 箇条書きでは句点（。）を入れないこと
  - 括弧の使い方は以下の通りとする
    - 括弧内に日本語が含まれる場合は全角括弧を使うこと
    - 英数字のみの場合は半角括弧を使うこと
  - コマンドの説明は以下の形式で記載すること
    - 説明文とコマンドをセットで記載する
    - 例
      - 開発サーバー起動

        ```sh
        bun run dev
        ```
